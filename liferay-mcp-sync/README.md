# Liferay MCP Sync

Utility scripts for syncing Liferay marketplace catalogs and skill definitions. Not an MCP server — these are data sync scripts used to populate MCP tool registries with Liferay ecosystem data.

## Scripts

- **index.js** — Main entry point
- **catalog-sync.js** — Sync Liferay marketplace catalog
- **master-sync.js** — Master sync orchestrator
- **extra-sync.js** — Sync additional metadata
- **skills-sync.js** — Sync skill definitions
- **official-skills-sync.js** — Sync official Liferay skills
- **marketplace.html** — Marketplace data reference

## Usage

```bash
npm install
npm start
```
